STEP 1: Start Docker Desktop
───────────────────────────────────────
  1. Press Cmd+Space
  2. Type: Docker
  3. Press Enter
  4. Wait until whale icon is steady (top menu bar)

STEP 2: Run Setup Script
───────────────────────────────────────

───────────────────────────────────────

  2. Run: npm run start:dev

  # Start database
  docker compose up -d postgres
  
  # Wait 10 seconds

  DB_HOST=localhost
  DB_PORT=5432
